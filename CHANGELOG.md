# Changelog

## 1.0.0

- Initial public release
- Arctic Sands dark theme with full UI styling (activity bar, tabs, sidebar, command palette, menus, hover widgets)
- Animated background image injection into VS Code editor
- Dashboard sidebar with ambient video background
- Live date/time display
- Weather widget with auto-detect location (IP geolocation) or manual city setting
- Cross-platform Spotify detection (Windows, macOS, Linux)
- Configurable settings: weather city, dashboard toggle, background opacity
- One-command install scripts for Windows (PowerShell) and Unix (bash)
